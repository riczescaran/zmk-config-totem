# Keymap Drawer

This folder contains keymap visualization files automatically generated by [keymap-drawer](https://github.com/caksoylar/keymap-drawer).

## Files

- **keymap_drawer.config.yaml** - Configuration file for keymap-drawer styling and keycodes
- **totem.svg** - Visual representation of the keyboard layout (auto-generated)
- **totem.yaml** - Parsed keymap data (auto-generated)

## How It Works

The GitHub Actions workflow automatically:
1. Parses the ZMK keymap files from `config/totem.keymap`
2. Generates SVG visualizations showing all layers
3. Commits the updated files back to this folder

The workflow triggers on:
- Changes to keymap files (`config/*.keymap`)
- Changes to the configuration (`keymap_drawer.config.yaml`)
- Manual workflow dispatch from GitHub Actions tab

## Viewing the Keymap

The generated SVG is displayed in the main [README](../readme.md) under the KEYMAP section.
